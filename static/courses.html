Learn DevOps from Basics

DevOps is a set of practices that combines software development (Dev) and IT operations (Ops).

It aims to shorten the systems development life cycle and provide continuous delivery with high software quality. DevOps complements Agile software development, with several DevOps aspects stemming from Agile methodology.
